# Font + asset attributions

The Rechnung v2 renderer bundles the following third-party fonts. Both ship
under permissive licenses compatible with embedding in commercial PDFs.

## Anton — `anton-regular.ttf`

- **Source**: <https://github.com/googlefonts/AntonFont> (Google Fonts mirror)
- **Designer**: Vernon Adams, The Anton Project Authors
- **License**: SIL Open Font License, Version 1.1
- **Use**: substitute for Impact in the "RECHNUNG" wordmark.

## DejaVu Sans — `dejavu-sans*.ttf`

- **Source**: DejaVu Fonts 2.37 (<https://dejavu-fonts.github.io>)
  Tarball: <https://downloads.sourceforge.net/project/dejavu/dejavu/2.37/dejavu-fonts-ttf-2.37.tar.bz2>
- **Designers**: DejaVu Fonts team (derived from Bitstream Vera Sans by
  Bitstream, Inc.; additional glyphs are public domain modifications)
- **License**: Bitstream Vera Fonts License (permissive, allows embedding +
  modification, no copyleft) + public-domain modifications. See
  <https://dejavu-fonts.github.io/License.html>
- **Use**: substitute for Verdana — used for all body, table, and footer text.
